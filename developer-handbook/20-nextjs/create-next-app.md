# Create Next.js App

## Purpose

Create a new Next.js project.

---

## Create Project

```bash
npx create-next-app@latest my-app 

Using TypeScript
npx create-next-app@latest my-app --typescript
Using PNPM
pnpm create next-app my-app
Start Development Server
npm run dev

Open:

http://localhost:3000
Project Structure
app/
components/
public/
lib/0
