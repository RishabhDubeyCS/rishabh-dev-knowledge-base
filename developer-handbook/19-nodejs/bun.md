# Bun

## Purpose

Modern JavaScript runtime, package manager, and bundler.

---

## Install Bun

### Linux/macOS

```bash
curl -fsSL https://bun.sh/install | bash
```

### Windows

```powershell
powershell -c "irm bun.sh/install.ps1 | iex"
```

---

## Check Version

```bash
bun -v
```

---

## Create Project

```bash
bun init
```

---

## Install Package

```bash
bun add express
```

---

## Run JavaScript

```bash
bun run app.js
```

---

## Run Development Server

```bash
bun run dev
```

---

## Why Bun?

```text
Fast startup
Fast package installs
Built-in TypeScript support
```
