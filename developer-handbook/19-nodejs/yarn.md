# Yarn

## Purpose

Alternative package manager for Node.js.

---

## Install Yarn

```bash
npm install -g yarn
```

---

## Initialize Project

```bash
yarn init -y
```

---

## Install Package

```bash
yarn add express
```

---

## Install Dev Dependency

```bash
yarn add -D typescript
```

---

## Run Script

```bash
yarn dev
```

---

## Cheat Sheet

```bash
yarn init -y
yarn add express
yarn add -D typescript
yarn dev
```
