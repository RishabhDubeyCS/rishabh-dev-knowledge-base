# PNPM

## Purpose

Fast and disk-efficient package manager.

---

## Install PNPM

```bash
npm install -g pnpm
```

---

## Initialize Project

```bash
pnpm init
```

---

## Install Package

```bash
pnpm add express
```

---

## Install Dev Dependency

```bash
pnpm add -D typescript
```

---

## Run Script

```bash
pnpm dev
```

---

## Why PNPM?

```text
Faster installs
Less disk usage
Better monorepo support
```

---

## Cheat Sheet

```bash
pnpm init
```
