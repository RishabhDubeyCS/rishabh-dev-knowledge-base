# npx

## Purpose

Run packages without installing them globally.

---

## Syntax

```bash
npx [package]
```

---

## Examples

Create Next.js app:

```bash
npx create-next-app@latest
```

Run TypeScript compiler:

```bash
npx tsc --init
```

Run Shadcn:

```bash
npx shadcn@latest init
```

---

## Why Use NPX?

```text
No global installation required.
Runs package instantly.
```
