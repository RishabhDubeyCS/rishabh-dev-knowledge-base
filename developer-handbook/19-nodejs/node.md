# Node.js

## Purpose

Run JavaScript outside the browser.

---

## Check Version

```bash
node -v
```

Output:

```bash
v24.5.0
```

---

## Run JavaScript File

```bash
node app.js
```

---

## Interactive Mode

```bash
node
```

Exit:

```bash
.exit
```

---

## Create First Program

```javascript
console.log("Hello Node.js");
```

Run:

```bash
node app.js
```

---

## Cheat Sheet

```bash
node -v
node app.js
node
```
