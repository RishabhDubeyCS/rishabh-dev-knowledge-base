# npm

## Purpose

Install and manage Node.js packages.

---

## Check Version

```bash
npm -v
```

---

## Initialize Project

```bash
npm init -y
```

---

## Install Package

```bash
npm install express
```

---

## Install Dev Dependency

```bash
npm install -D typescript
```

---

## Remove Package

```bash
npm uninstall express
```

---

## Run Script

```bash
npm run dev
```

---

## Cheat Sheet

```bash
npm init -y
npm install express
npm install -D typescript
npm uninstall express
npm run dev
```
