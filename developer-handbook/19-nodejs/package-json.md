# package.json

## Purpose

Store project metadata, dependencies, and scripts.

---

## Create package.json

```bash
npm init -y
```

---

## Example

```json
{
  "name": "my-app",
  "version": "1.0.0",
  "scripts": {
    "dev": "next dev",
    "build": "next build"
  }
}
```

---

## Run Scripts

```bash
npm run dev
npm run build
```

---

## Common Sections

```json
{
  "name": "",
  "version": "",
  "scripts": {},
  "dependencies": {},
  "devDependencies": {}
}
```
