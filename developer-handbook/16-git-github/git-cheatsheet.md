# Git Cheat Sheet

## Daily Commands

```bash
git status
git add .
git commit -m "message"
git push
git pull
```

## Branching

```bash
git branch
git checkout -b feature
git merge feature
```

## Stash

```bash
git stash
git stash pop
```

## Undo

```bash
git reset file.txt
git reset --soft HEAD~1
```

## Advanced

```bash
git rebase main
```
