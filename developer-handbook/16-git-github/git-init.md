# git init

## Purpose

Initialize a new Git repository.

---

## Syntax

```bash
git init
```

---

## Step 1: Create Project

```bash
mkdir my-project
cd my-project
```

---

## Step 2: Initialize Git

```bash
git init
```

Output:

```text
Initialized empty Git repository
```

---

## Step 3: Check Status

```bash
git status
```

---

## Step 4: Add Files

```bash
git add .
```

---

## Step 5: Commit

```bash
git commit -m "Initial commit"
```

---

## Cheat Sheet

```bash
git init
git status
git add .
git commit -m "Initial commit"
```
