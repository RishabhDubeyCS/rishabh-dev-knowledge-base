# git clone

## Purpose

Download an existing Git repository.

---

## Syntax

```bash
git clone <repository-url>
```

---

## Example

```bash
git clone https://github.com/user/project.git
```

---

## What Happens?

```text
1. Downloads code
2. Downloads commit history
3. Connects remote origin
```

---

## Verify

```bash
cd project
git remote -v
```

---

## Cheat Sheet

```bash
git clone <url>
git remote -v
```
